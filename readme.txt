== Description ==

Upload local images to Thumbr.io, create thumbnails and edit it.
Create thumbnails and apply vintage effects to your images with thumbr.io.

== Installation ==

1. Upload the folder `wordpress` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Once the plugin is installed, you have to go to Settings -> Thumbr.io.
4. Write your Api key and secret key and push in "Save changes"

== Frequently Asked Questions ==

= How can I upload an image to Thumbr.io?

You have two possible alternatives: 
	1. Go to Media->Thumbrio Uploader. You will be able to upload all files you want.
	2. Push the "Add media button" when you are editing a post. In the popout menu, click on
	   "Insert media Thumbr.io" and then on "Upload Files".
Once you have uploaded your files, you will be able to use these images and include them in your posts.

= How can I use the images which are located in thumbrio ?

You have two possible options:
	1. Go to Media -> Thumbrio Uploader. In that page it is showed up the 20 newest images. 
	You will be able select all the images you want to use them click on the button "Use images".

	2. Push "Add Media Button" when you are editing a post. Click on "Insert media Thumbr.io" and then on "Media Library".
	In this menu, you will be able to see all your images in Thumbr.io. To use an image, you have to select all the images
	you want and press the button "Use images".

To show all these images, a connection to amazon.s3 is performed. The response to all these connections are saved but for only 5 minutes.

= How can I see all the images I have uploaded to Thumbr.io?

Go to Media -> Library, you'll see all your pictures. If you click on a image, you'll be able to edit it. This means: 
Choose an effect; change the hue, saturation, and lightness levels; change the brightness, contrast and gamma levels;
and rotate the image.

= What do I need to run this plugin?

You need to set your storage backend settings in http://www.thumbr.io/profile/storage. This plugin only works
for a Thumbr.io and a Amazon s3 backend, NOT for a Dropbox backend.
Once you have set the storage backend, you have to go to Settings->Thumbr.io and write your api key and secret key of Thumbr.io
